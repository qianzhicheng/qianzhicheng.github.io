弄了两天终于把这个 blog 弄好了，先来说说步骤。安装 git -> 安装 Ruby -> 安装 RubyDevKit -> 安装 jekyll -> 安装 bundle -> 选 jekyll 模板 -> 编辑 _conf ig.yml -> 本地测试 -> 上传 git 。不得不说一句，windows 真不适合做开发，配置环境实在是麻烦。

git 和 rubyinstaller 除了下载速度有点慢以外，安装倒还无脑。Devkit 就怎么都识别不了 ruby 路径，最后发现64位的 ruby 很多人都遇到了这个问题，解决办法试了很多，对我来说唯一有效的是卸载64位的 ruby 安装32位的，devkit 也换成32位。

之后有时间写写如何在 git 上创建个人博客，我就以个人经历为例吧。先挖个坑，周末再填。

先给出几个所需软件的下载地址（Windows 下）：

git：git for windows.


ruby&devkit: rubyinstaller for windows. （devkit 注意和 ruby 版本要一致）


jekyll：jekyll theme.
